ChartDirector Ver 5.0 (Perl Edition)

Zoomable and Scrollable Charts

This section describes the ChartDirector Zooming and Scroll Framework for creating charts system extremely wide range of zoom ratios and rich user controls.

In this section

Zooming and Scrolling Framework
Description of the ChartDirector Zooming and Scroll Framework.

Zooming and Scrolling Demonstration
This sample program demonstrates a zoomable and scrollable chart for time based data series, with drag to scroll, drag to zoom, click to zoom, as well as allowing the start and end dates to selected directly using drop down list boxes. The chart supports both partial updates (AJAX updates) and full page updates.

Zooming and Scrolling Demonstration (2)
This sample program demonstrates a general zoomable and scrollable chart, with drag to scroll, drag to zoom and click to zoom. The view port can also be moved by dragging on the rectangle on the navigation pad. The chart supports both partial updates (AJAX updates) and full page updates.
