Meters and Gauges

In this section, we will discuss how to create meters and gauges using ChartDirector through a number of examples.

TitleDescription
Semi-Circle MeterDemonstrates a simple half circle angular meter.
Round MeterDemonstrates a round meter.
Wide Angular MetersDemonstrates angular meters of which the width is large compared to the height. Also demonstrates various background effects.
Square Angular MetersDemonstrates angular meters of which the pointers are pivoted at the corners of squares.
Multi-Pointer Angular MeterDemonstrates adding multiple pointers of different styles to a meter.
Icon Angular MeterDemonstrates a fuel-tank like meters, in which the meter is labelled by an icon and the scale shows text abbreviations.
Horizontal Linear MeterDemonstrates a simple horizontal linear meter.
Vertical Linear MeterDemonstrates a simple vertical linear meter.
Multi-Pointer Horizontal MeterDemonstrates a horizontal linear meter with multiple pointers.
Multi-Pointer Vertical MeterDemonstrates a vertical linear meter with multiple pointers.
Linear Zone MeterDemonstrates a linear meter which is labelled with zones.
