WebChartViewer.getZoomOutRatio

Usage

getZoomOutRatio()

Description

Gets the magnification ratio for each zoom out click of the mouse.

For zoom out, the ratio should be less than 1. The default is 0.5.

Arguments

None

Return Value

The magnification ratio.
