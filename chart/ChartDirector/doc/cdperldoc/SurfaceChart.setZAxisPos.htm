SurfaceChart.setZAxisPos

Usage

setZAxisPos(pos)

Description

Sets the position of the z-axis.

In a surface chart, the z-axis can be drawn either on the left side or the right side. By default, ChartDirector will draw the z-axis on the side that is nearer to the viewer.

This method can be used to specify which side the z-axis should be drawn.

Arguments

ArgumentDefaultDescription
pos(Mandatory)The position of the z-axis, which must be either Left or Right.

Return Value

None
