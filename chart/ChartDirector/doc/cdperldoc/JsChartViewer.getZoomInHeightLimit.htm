JsChartViewer.getZoomInHeightLimit

Usage

getZoomInHeightLimit()

Description

Gets the view port height at maximum zoom in for mouse zoom in actions.

Arguments

None

Return Value

The view port height at maximum zoom in. See JsChartViewer.setZoomInHeightLimit for the meaning of this value.
